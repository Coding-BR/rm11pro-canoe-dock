# Security Policy

This project documents boot-chain modification for a specific device family. Treat every artifact as dangerous until its model, firmware baseline, hash, and rollback path are verified.

Do not open public issues containing:

- IMEI, serial number, phone number, account identifiers, or personal files.
- Full private partition dumps.
- Calibration partitions such as `persist`, `modemst1`, `modemst2`, or `fsg`.
- Secrets, tokens, or private keys.

For public reports, redact personal identifiers and share only the minimum command output needed to reproduce the result.
