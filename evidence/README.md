# Evidence

Evidence folders index proof files and public-safe summaries.

Do not store:

- Private partition images.
- Raw full dumps.
- IMEI, serials, phone numbers, or account identifiers.
- Paid/proprietary firmware.

Use:

- `kernel/` for AnyKernel, KernelSU-Next, SUSFS, root-state, and boot proof.
- `recovery/` for OrangeFox and stock recovery rollback proof.
- `gsi/` for GSI/ROM reports.
