# Contributing

This repo accepts evidence-first updates.

Good contributions include:

- Exact device model and firmware baseline.
- Exact artifact filename and SHA-256.
- Commands used.
- Redacted command output.
- Boot, reboot, hardware, idle, and rollback result.
- Clear broken-feature notes.

Do not submit:

- Private partition dumps.
- Paid/proprietary firmware.
- Other users' calibration partitions.
- Large release ZIPs or images unless a maintainer explicitly requests them.
- Claims that are not backed by reproducible evidence.
