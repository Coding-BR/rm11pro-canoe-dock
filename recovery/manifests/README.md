# Recovery Manifests

Recovery manifests record hashes and source maps used by the OrangeFox lane.

Tracked here:

- `d2n-baseline.sha256`: frozen D2N image and zip hashes.
- `android16-live-prebuilts.sha256`: hash manifest for the local Android 16
  prebuilt capture used during recovery experiments.

Raw images, zips, and private prebuilt payloads stay outside git unless they
are intentionally published as release artifacts.
